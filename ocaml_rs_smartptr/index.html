ptr
This module provides the DynBox smart pointer, which is a wrapper around the registry’s DynArc with PhantomData for type safety. DynBox allows the user to wrap the object in a Mutex or shared RwLock. By default, using .into() will create a Mutex-protected version (exclusive).
registry
This module provides a Registry for managing type coercions and type information in a Rust-OCaml binding context. The Registry allows for the registration of coercion functions that convert between different types, leveraging Rust’s type system and OCaml’s garbage collector.
