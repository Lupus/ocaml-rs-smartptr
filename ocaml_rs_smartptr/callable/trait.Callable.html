Expand description

The Callable trait represents a function or closure that can be called with a set of arguments to produce a return value. This trait is designed to be used with OCaml values and provides methods for calling the function, describing its arguments, and generating unique IDs for the function signature.

§Type Parameters

  • Ret: The return type of the function, which must implement ocaml::FromValue and OCamlDesc.
