Expand description

A trait that is implemented by P1, P2, etc., used as a link between concrete P1, P2, etc., and the WithTypeParams wrapper type below.

Required Methods§

Source

fn params_string() -> String

Returns a string representation of the type parameters.

Source

fn params_count() -> usize

Returns the count of type parameters.

Dyn Compatibility§

This trait is not dyn compatible.

In older versions of Rust, dyn compatibility was called "object safety", so this trait is not object safe.
